# Melody Generator

Проект для генерации простой музыкальной мелодии на основе заданной тональности, количества тактов и темпа.

## Цель проекта

Создать инструмент, который позволит программно формировать музыкальные последовательности и сохранять их для последующего прослушивания или анализа.

## Структура проекта

```
project/
│
├── src/
│   ├── main.py                # Точка входа
│   │
│   ├── entities/              # Основные сущности проекта
│   │   ├── note.py            # Нота
│   │   ├── scale.py           # Гамма
│   │   ├── melody.py          # Мелодия
│   │
│   └── services/             # Логические компоненты
│       ├── generator.py       # Генерация мелодии (будет реализована позже)
│       ├── exporter.py        # Экспорт в MIDI (будет реализован позже)
│       └── visualizer.py      # Визуализация мелодии (будет реализована позже)
│
├── requirements.txt
└── .gitignore
```
